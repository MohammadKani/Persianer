## پرشینر چیه؟ 🤔

پرشینر یک افزونه سبک برای Chrome است که صفحه های وب را برای کاربر فارسی زبان راحت تر می کند.

کار اصلی آن:

- 📅 تاریخ میلادی را هرکجا که باشه به شمسی (جلالی) تبدیل می کند.
- 📝 متن های فارسی را هوشمندانه راست چین می کند و فونت مناسب میده.
- 📊 حتی در اکثر چارت ها و نمودار ها کار میکنه.
- ⚡ روی همان تب فعال کار می کند و استفاده از آن خیلی ساده است.


---


## امکانات اصلی 🌟

### 1) تبدیل تاریخ میلادی به شمسی 📆

نمونه هایی که تشخیص می دهد:

- `2024-11-15`
- `2024-11-15 14:30:00`
- `11/15/2024`
- `Sep`
- `November 15, 2024`

### 2) راست چین خودکار فارسی 🧭

- وقتی متن فارسی باشد، جهت را راست چین می کند.
- اگر فقط این قابلیت را بخواهید، مستقل روشن می ماند.

### 3) تنظیمات ساده 🎛️

- دکمه روشن/خاموش تبدیل تاریخ
- دکمه روشن/خاموش راست چین
- نمایش وضعیت فعال یا غیرفعال به صورت واضح

---

## نصب آسان 🚀

1. نسخه آخر را از اینجا دانلود کن و از حالت فشرده خارج کن.
2. در Chrome بروید به: `chrome://extensions/`
3. گزینه **Developer mode** را روشن کنید.
4. روی **Load unpacked** بزنید.
5. پوشه پروژه Persianer را انتخاب کنید.

تمام! حالا آیکون افزونه را بزنید و قابلیت ها را روشن کنید ✅

---

## روش استفاده خیلی ساده (برای همه) 🧩

1. وارد یک سایت شوید که تاریخ میلادی دارد.
2. روی آیکون افزونه کلیک کنید.
3. گزینه های دلخواه را فعال کنید.
4. همان تب فعال، رفرش می شود و نتیجه را می بینید.

پیشنهاد:

- اگر فقط تاریخ می خواهید: فقط گزینه تاریخ روشن باشد.
- اگر فقط راست چین می خواهید: فقط گزینه راست چین روشن باشد.
- اگر هر دو را می خواهید: هر دو گزینه روشن باشند.

## English (Quick Intro)

Persianer is a beginner-friendly Chrome extension for Persian users.

- Converts Gregorian dates to Jalali
- Applies smart RTL to Persian text blocks
- Uses simple popup toggles
- Runs fully local with no tracking

Install quickly:

1. Open Chrome extensions page: `chrome://extensions/`
2. Enable Developer mode
3. Load unpacked and select this project folder
